## req

1 homebrew

     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2 install git 
  ```
brew install git
```
3 clone this repo 
  ```
https://github.com/matbagilcz/matbook.git
```

4 cd into directory
  ```
  cd matbook
```

5 install apps from brewfile
  ```
brew bundle install
```
